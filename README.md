# Shyam Charan — Portfolio

Personal research portfolio for [Shyam Charan Kesavamoorthi](https://inclinedtaco.github.io/), focused on robot learning, loco-manipulation, and multi-humanoid systems.

## Development

```bash
npm install
npm run dev
```

The site uses Next.js, React, Motion, and a static export for GitHub Pages. Pushes to `main` build and deploy automatically.

## Design

The interaction restraint and single-page pacing are inspired by [ibelick/nim](https://github.com/ibelick/nim). The layout, robotics visual system, content model, styling, and implementation in this repository are custom.
